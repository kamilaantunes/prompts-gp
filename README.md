# Prompts para Gerar Auxiliar em Documentações

## 1. Plano de Gerenciamento do Projeto (PGP)

> Crie um Plano de Gerenciamento do Projeto formatado que integre o escopo, os objetivos e as restrições de [nome do projeto], incluindo todas as secções necessárias para uma orientação abrangente. O objetivo do projeot é [objetivo do projeto].

## 2. Estrutura Analítica do Projeto (EAP)

> Gere uma Estrutura Analítica do Projeto detalhada para [tipo ou nome do projeto], decompondo o escopo em componentes gerenciáveis para facilitiar o acompanhamento e o controle.

## 3. Cronograma  - Gráfico de Gantt

> Crie um esboço de Gráfico de Gantt para [nome do projeto], detalhando o cronograma e marcando os marcos críticos ao longo da duração do projeto.

- Softwares a utilizar: MS Project, Asana, Monday etc.

## 4. Plano de Recursos

> Formule um Plano de Recursos para [nome do projeto], identificando todos os recursos necessários e propondo uma estratégia para sua alocação e utilização dentro dos prazos previstos.

## 5. Plano de Gerenciamento de Riscos

> Desenvolva um Plano de Gerenciamento de Riscos para [nome do projeto], identificando os riscos potenciais, seus impactos e as estratégias de mitigação para projeto o projeto.

- Categorar os riscos por probabilidade e impacto.
- Matriz de riscos funcional com esforço adicional.

## 6. Plano de Engajamento de Stakeholders

> Elaborar um Plano de Engajamento de Partes Interessadas para [nome do projeto], especificando métodos e estratégias para uma comunicação eficaz e o envolvimento de todas as partes interessadas.

- Incluir os principais stakeholders do projeto.

## 7. Plano de Gerenciamento de Custos

> Costrua um Plano de Gerenciamento de Custos detalhado para [nome do projeto], incluindo estimaticas de orçamento e mecanismos de controle para garantir eficiência financeira e transparência.
